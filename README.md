# VSCode Extensions for Laravel

## How to install

```sh
# Fork/Clone this repository
git clone alphaolomi/recommended-plugins-for-laravel
# Open the repository in VSCode

# A Notification will prompt you to install the extensions

# Happy Coding!
```
